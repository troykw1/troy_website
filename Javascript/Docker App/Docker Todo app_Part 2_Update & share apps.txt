3. For the repo name, enter getting-started. Make sure that the Visibility is Public.
4. Select Create.

This section gives an example command to run to push to this repo.
--> docker push troykw1/getting-started:tagname

7. Use the following command to give the getting-started image a new name.

=== Run the image on a new instance === 

1. Open your browser to Play with Docker.
https://labs.play-with-docker.com/

2. Sign in with your Docker Hub account.
3. Select Start and then select the + ADD NEW INSTANCE
4. In the terminal, start your app.
