2) To confirm your list of database use 
\l

3) Now we'll tell our laptop on cmd we want to use the test database
 \c test
 
4) Now create a table called "users" with columns id, name and email
---> I inserted this into the commnad line 

CREATE TABLE users (
    id SERIAL PRIMARY KEY,
    name VARCHAR(255) NOT NULL,
    email VARCHAR(255) UNIQUE NOT NULL
);

\l  Lists all available databases on the server 
\c <database_name>: Switches to the specified database 

5) Now we seed a dummy data into the table
---> I inserted this into the commnad line 

INSERT INTO users (name, email)

    ('John Rather', 'john.rather@example.com'),
    ('Jane Smith', 'jane.smith@example.com');
